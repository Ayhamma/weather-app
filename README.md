# Приложение для прогноза погоды

ФИО:Маатук Айхам
Группа: К 3321

## Описание

Веб-приложение прогноза погоды, реализованное без фреймворков (чистые HTML, CSS и JavaScript).

Функциональность:
- При первом открытии запрашивается геолокация пользователя.
- При разрешении геолокации отображается прогноз для «Текущего местоположения» (сегодня + минимум 2 дня).
- При отказе от геолокации пользователь может выбрать город из списка с автодополнением.
- Можно добавлять дополнительные города (минимум 2) и переключаться между ними.
- Есть кнопка **«Обновить прогноз»**, которая выполняет новые HTTP-запросы к API погоды (без перезагрузки страницы).
- Состояние (список городов и выбранный город) сохраняется в `localStorage` и восстанавливается после перезагрузки страницы.

## Технологии

- HTML5
- CSS3 (адаптивная верстка, без CSS-фреймворков)
- JavaScript (без React/Vue/Angular)
- [Open-Meteo API](https://open-meteo.com/) для получения прогноза погоды
